# 画像配置ガイド

このフォルダに以下のファイル名で画像を配置すると、`index.html` から自動で読み込まれます。
画像が無い場合は、ハッチング柄のプレースホルダーが表示されます。

| ファイル名 | 用途 | 推奨サイズ・形式 |
| --- | --- | --- |
| `logo.png` | ヘッダー・フッターのロゴマーク | 200×200 / 透過PNG |
| `hero.jpg` | メインビジュアル（ファーストビュー） | 800×1000（縦長 4:5） |
| `empathy.jpg` | お悩みセクションの写真 | 700×700（正方形） |
| `before.jpg` | 改善事例 BEFORE | 600×600（正方形） |
| `after.jpg` | 改善事例 AFTER | 600×600（正方形） |
| `voice-01.jpg` 〜 `voice-03.jpg` | お客様の声 顔写真 | 200×200（正方形） |
| `feature-01.jpg` 〜 `feature-04.jpg` | 当院の特徴 写真 | 600×450（4:3） |
| `cta-bg.jpg` | CTAセクションの背景写真 | 1600×900（16:9） |
| `store-01.jpg` 〜 `store-05.jpg` | 各店舗の外観/内観 | 1600×900（16:9） |

## 差し替え方法
1. 上記のファイル名でこのフォルダに画像を保存するだけ。
2. ファイル名を変えたい場合は `index.html` 内の `<img src="images/xxx.jpg">` のパスを書き換えてください。
3. 画像が表示されない場合は、ファイル名・拡張子（大文字小文字）を確認してください。
